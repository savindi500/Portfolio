# My Portfolio Website

## Description
This is my personal portfolio website built using React.js. It showcases my skills, projects, and experiences in software development. The website is fully responsive and includes animations for an engaging user experience.

## Features
- ✅ Responsive Design
- ✅ Smooth Animations
- ✅ Interactive UI
- ✅ Dynamic Skills & Projects Section
- ✅ Contact Form

## Tech Stack
- **Frontend:** React.js, Material-UI
- **Other Tools:** Framer Motion
